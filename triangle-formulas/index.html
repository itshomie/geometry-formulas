Direct answer

What are the main triangle formulas?

The main triangle formulas are area A=1/2bh, perimeter P=a+b+c, and the right-triangle formula a2+b2=c2. Use base and perpendicular height for area, side lengths for perimeter, and the Pythagorean theorem only for right triangles.

Short answer

Triangle formulas at a glance

Area: A=21​×b×h Perimeter: P=a+b+c Right triangle: a2+b2=c2

Definition

What is a triangle?

A triangle is a 2D polygon with three sides, three vertices, and interior angles that add up to 180 degrees.

Formula explanation

How to use these formulas

Use base and height when the height is known. Use side lengths for perimeter. Use the Pythagorean theorem only for right triangles.

Examples

Step-by-step examples

Find triangle area

  1. Given b = 10 cm and h = 6 cm.
  2. Use A=1/2bh.
  3. A=21​×10×6=30cm2.

Find perimeter

  1. Given sides 4, 5, and 7 m.
  2. Use P=a+b+c.
  3. P=4+5+7=16m.

Right triangle hypotenuse

  1. Given legs 3 and 4.
  2. Use a2+b2=c2.
  3. c=9+16​=5.

Find height from area

  1. Given A = 48 cm2 and b = 12 cm.
  2. Use h=2A/b.
  3. h=2×48/12=8cm.

Equilateral triangle area

  1. Given s = 6 m.
  2. Use A=(3​/4)s2.
  3. A=93​≈15.59m2.

Common mistakes

Common mistakes

  • Using a slanted side as height. Height must be perpendicular to the base.
  • Using the Pythagorean theorem on a triangle that is not right.
  • Forgetting to check the triangle inequality before using three side lengths.

FAQ

Triangle Formulas FAQ

What is the basic triangle area formula?

The basic formula is A=21​×b×h.

What is the perimeter of a triangle?

Add the three side lengths: P=a+b+c.

When do I use the Pythagorean theorem?

Use it only for right triangles, where c is the hypotenuse opposite the right angle.
