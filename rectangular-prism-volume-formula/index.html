Direct answer

What is the rectangular prism volume formula?

The rectangular prism volume formula is V=lwh, where l is length, w is width, and h is height. It is also the box volume formula and uses cubic units.

Method

How to use the formula

  1. Check that length, width, and height use the same unit.
  2. Multiply the three dimensions using V=lwh.
  3. For reverse problems, divide volume by the product of the two known dimensions.
  4. Write the answer with cubic units for volume.

Concept

Understanding rectangular prism volume formula

This guide focuses on one calculation: volume from dimensions. The main formula is V=lwh. Use this page when the problem is asking for that specific measurement, and use the related links when the problem is really asking for a broader shape review or a different geometry formula.

Before substituting numbers, identify each known measurement and match it to the variable in the formula. Most wrong answers come from using the right formula with the wrong input, such as using diameter where radius is required, using slant height where vertical height is required, or treating a side length as an area.

Because this is a volume formula, the final answer should use cubic units. If the measurements are in centimeters, the volume is in cubic centimeters; if the measurements are in feet, the volume is in cubic feet. Convert mixed units before substituting values so the multiplication does not combine different scales.

The reverse formulas are included for problems that give the result first and ask for a missing measurement. On this page, those reverse tasks include height from volume, width from volume, length from volume. Use a reverse formula only when the known values match the formula table; otherwise, go back to the main shape page and choose a different formula.

After calculating, estimate whether the answer is reasonable. If every input gets larger, area and volume should usually get larger too. If a reverse calculation gives a negative value or a value that is larger than the original measurement in an impossible way, recheck the substituted values, the unit conversion, and the order of operations.

Examples

Step-by-step examples

Volume from dimensions

  1. Given l = 9 cm, w = 5 cm, h = 4 cm.
  2. Use V=lwh.
  3. V=9×5×4=180cm3.

Height from volume

  1. Given V = 240 ft3, l = 12 ft, w = 5 ft.
  2. Use h=V/lw.
  3. h=240/60=4ft.

Width from volume

  1. Given V = 144 in3, l = 8 in, h = 6 in.
  2. Use w=V/lh.
  3. w=144/48=3in.

Common mistakes

Common mistakes

  • Adding dimensions instead of multiplying them.
  • Mixing units before multiplying.
  • Writing volume in square units.
