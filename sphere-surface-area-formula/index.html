Direct answer

What is the sphere surface area formula?

The sphere surface area formula is SA=4πr2, where r is radius. Surface area uses square units because it measures the outside area, not the space inside.

Method

How to use the formula

  1. Convert diameter to radius if needed.
  2. Square the radius, then multiply by 4π.
  3. Use square units for the answer.
  4. For reverse problems, divide surface area by 4 pi and take the square root.

Concept

Understanding sphere surface area formula

This guide focuses on one calculation: surface area from radius. The main formula is SA=4πr2. Use this page when the problem is asking for that specific measurement, and use the related links when the problem is really asking for a broader shape review or a different geometry formula.

Before substituting numbers, identify each known measurement and match it to the variable in the formula. Most wrong answers come from using the right formula with the wrong input, such as using diameter where radius is required, using slant height where vertical height is required, or treating a side length as an area.

Because this is a surface area formula, the final answer should use square units. Surface area measures the outside of a three-dimensional solid, so it is not written in cubic units. A quick unit check catches many mistakes before you compare the numerical answer.

The reverse formulas are included for problems that give the result first and ask for a missing measurement. On this page, those reverse tasks include surface area from diameter, radius from surface area, diameter from surface area. Use a reverse formula only when the known values match the formula table; otherwise, go back to the main shape page and choose a different formula.

After calculating, estimate whether the answer is reasonable. If every input gets larger, area and volume should usually get larger too. If a reverse calculation gives a negative value or a value that is larger than the original measurement in an impossible way, recheck the substituted values, the unit conversion, and the order of operations.

For more context, compare this focused guide with Sphere Formulas, Sphere Volume Formula, Surface Area Formulas. Those pages are linked below so this page can stay centered on sphere surface area formula while still giving you a path to nearby formulas.

Examples

Step-by-step examples

Surface area from radius

  1. Given r = 7 cm.
  2. Use SA=4πr2.
  3. SA=196π≈615.75cm2.

Surface area from diameter

  1. Given d = 16 ft.
  2. First use r=d/2, so r = 8 ft.
  3. SA=256π≈804.25ft2.

Radius from surface area

  1. Given SA = 100π in2.
  2. Use r=SA/4π​.
  3. r=25​=5in.

Common mistakes

Common mistakes

  • Using r cubed instead of r squared.
  • Using diameter as radius.
  • Writing surface area in cubic units.
