Direct answer

What is the triangle area formula?

The triangle area formula is A=1/2bh, where b is the base and h is the perpendicular height. The height must meet the base at a right angle; a slanted side is not the height unless it is perpendicular to the chosen base.

Method

How to use the formula

  1. Choose a base side.
  2. Use the perpendicular height to that base.
  3. Multiply base by height, then divide by 2 using A=1/2bh.
  4. Write the answer with square units.

Concept

Understanding triangle area formula

This guide focuses on one calculation: area from base and height. The main formula is A=1/2bh. Use this page when the problem is asking for that specific measurement, and use the related links when the problem is really asking for a broader shape review or a different geometry formula.

Before substituting numbers, identify each known measurement and match it to the variable in the formula. Most wrong answers come from using the right formula with the wrong input, such as using diameter where radius is required, using slant height where vertical height is required, or treating a side length as an area.

Because this is an area formula, the final answer should use square units. Area measures flat space, so a measurement in inches produces square inches and a measurement in meters produces square meters. If the problem mixes units, convert first and then apply the formula.

The reverse formulas are included for problems that give the result first and ask for a missing measurement. On this page, those reverse tasks include height from area and base, base from area and height, heron's formula. Use a reverse formula only when the known values match the formula table; otherwise, go back to the main shape page and choose a different formula.

After calculating, estimate whether the answer is reasonable. If every input gets larger, area and volume should usually get larger too. If a reverse calculation gives a negative value or a value that is larger than the original measurement in an impossible way, recheck the substituted values, the unit conversion, and the order of operations.

For more context, compare this focused guide with Triangle Formulas, Area Formulas, Pythagorean Theorem Formula. Those pages are linked below so this page can stay centered on triangle area formula while still giving you a path to nearby formulas.

Examples

Step-by-step examples

Area from base and height

  1. Given b = 12 cm and h = 7 cm.
  2. Use A=1/2bh.
  3. A=21​×12×7=42cm2.

Height from area

  1. Given A = 60 ft2 and b = 15 ft.
  2. Use h=2A/b.
  3. h=120/15=8ft.

Base from area

  1. Given A = 45 m2 and h = 9 m.
  2. Use b=2A/h.
  3. b = 90 / 9 = 10 m.

Common mistakes

Common mistakes

  • Using a side length as height when it is not perpendicular.
  • Forgetting to divide by 2.
  • Using linear units for area.

FAQ

Triangle Area Formula FAQ

What is the formula for the area of a triangle?

The formula is A=1/2bh.

Does the triangle height have to be perpendicular?

Yes. Height is the perpendicular distance from the base to the opposite vertex.

Can I find height from area?

Yes. Rearrange the formula to h=2A/b.
