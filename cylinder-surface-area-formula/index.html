Direct answer

What is the cylinder surface area formula?

The total cylinder surface area formula is TSA=2πr(r+h). Lateral surface area is only the curved side, LSA=2πrh. Total surface area includes the curved side plus both circular bases.

Method

How to use the formula

  1. Decide whether the problem asks for total surface area or lateral surface area.
  2. Use LSA=2πrh for the curved side only.
  3. Add two circular bases with 2πr2 for total surface area.
  4. Write the answer with square units.

Concept

Understanding cylinder surface area formula

This guide focuses on one calculation: total surface area. The main formula is TSA=2πr(r+h). Use this page when the problem is asking for that specific measurement, and use the related links when the problem is really asking for a broader shape review or a different geometry formula.

Before substituting numbers, identify each known measurement and match it to the variable in the formula. Most wrong answers come from using the right formula with the wrong input, such as using diameter where radius is required, using slant height where vertical height is required, or treating a side length as an area.

Because this is a surface area formula, the final answer should use square units. Surface area measures the outside of a three-dimensional solid, so it is not written in cubic units. A quick unit check catches many mistakes before you compare the numerical answer.

The reverse formulas are included for problems that give the result first and ask for a missing measurement. On this page, those reverse tasks include lateral surface area, one base area, expanded total area. Use a reverse formula only when the known values match the formula table; otherwise, go back to the main shape page and choose a different formula.

After calculating, estimate whether the answer is reasonable. If every input gets larger, area and volume should usually get larger too. If a reverse calculation gives a negative value or a value that is larger than the original measurement in an impossible way, recheck the substituted values, the unit conversion, and the order of operations.

For more context, compare this focused guide with Cylinder Formulas, Cylinder Volume Formula, Surface Area Formulas. Those pages are linked below so this page can stay centered on cylinder surface area formula while still giving you a path to nearby formulas.

Examples

Step-by-step examples

Total surface area

  1. Given r = 3 cm and h = 8 cm.
  2. Use TSA=2πr(r+h).
  3. TSA=2π×3(3+8)=66π≈207.35cm2.

Lateral surface area

  1. Given r = 3 cm and h = 8 cm.
  2. Use LSA=2πrh.
  3. LSA=48π≈150.80cm2.

Area of two bases

  1. Given r = 3 cm.
  2. Use 2B=2πr2.
  3. 2B = 18π ≈ 56.55 cm2.

Common mistakes

Common mistakes

  • Using lateral surface area when the problem asks for total surface area.
  • Forgetting the two circular bases.
  • Using cubic units for surface area.
